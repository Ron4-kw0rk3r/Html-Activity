HT-Lights Conocimientos sobre Web

HTML: Hyper Text Markup Lenguaje es el lenguaje estándar para crear páginas web, este lenguaje de etiquetas permite crear el cuerpo de una pagina web.


HTML utiliza etiquetas para estructurar el contenido.


Las etiquetas HTML pueden tener atributos que proporcionan información adicional.


HTML5 introdujo nuevas etiquetas y funcionalidades.

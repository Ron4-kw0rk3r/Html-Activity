Hola, esta página se creó con el propósito de aprender de manera breve sobre HTML, el cual es el lenguaje que, con la ayuda de un archivo de formato HTML, permite crear una base de página en esqueleto para una más formalizada. En pocas palabras, esta demo de página emplea pasos simples desde la elaboración de un control mínimo hasta uno real más completo y así hasta superarse.
